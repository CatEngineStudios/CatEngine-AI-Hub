API AI Chat – Usage Guide & Privacy Policy

1. What This App Does

API AI Chat is a web-based tool that allows you to connect to different AI models (such as OpenAI ChatGPT, Gemini, DeepSeek, etc.) using your own API keys. We do not provide API keys — you must obtain them from the official AI provider.

2. How to Use

  1. Obtain an API key from the provider you want to use (e.g., OpenAI, Google Gemini, DeepSeek).
  2. Paste your API key into the field provided in the sidebar.
  3. Select your preferred AI model from the dropdown menu.
  4. Start chatting — your messages will be sent directly to the provider's API.
  5. Your chat history and API keys are stored locally in your browser for convenience.
Important: Do not share your API key with anyone. Keep it private and secure. Only use this app on devices and browsers you trust.

3. Data Storage & Privacy

We respect your privacy — here’s how your data is handled:

  • We do NOT collect, store, or transmit any of your chat data or API keys to our servers.
  • All data (API keys, chat history) is stored locally in your browser using localStorage or sessionStorage.
  • Because storage is local, if you clear your browser data or use a different device/browser, your chats and keys will be lost.

4. Security Risks

  • API keys stored in your browser can be accessed if your device is compromised or if malicious code is injected into the site.
  • Never use your API key on untrusted networks or public/shared computers.
Tip: If you are concerned, use a temporary/incognito session so your key is never saved after you close the browser.

5. Disclaimer

This app is provided "as is", without any warranties of any kind. Use at your own risk. We are not responsible for:

  • Loss of API keys or unauthorized use of your key.
  • Any charges incurred on your account from the AI provider.
  • Loss of data due to browser cache clearing or device issues.
