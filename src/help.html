CatEngine AI Hub — Help & Privacy

Welcome

This page explains how CatEngine AI Hub works, what’s new, and how your data is handled in Guest vs Google Sign-In modes.

What’s New

  • Multi-provider models: OpenAI (incl. GPT-5 / GPT-4 series), DeepSeek, and Gemini.
  • Streaming output: see replies arrive token-by-token; a Stop button cancels mid-generation.
  • Attachments: add .txt, .md, .json, .js, .py, .java, .cs and more. Images work on vision-capable models.
  • Auto chat titles: first user message generates a short, generic title automatically.
  • Chat management: rename, delete, import into chat, and export via the chat’s ⋯ menu.
  • Profile page: toggle Dark Mode, pick default model, manage key, clear chats, switch account.
  • Sliding context window: trims oldest turns to avoid token-limit errors.

Guest Mode

No account required. Your data is only saved on this device/browser.

  • Storage: API key and chats live in your browser (e.g. localStorage).
  • Cloud: Nothing is uploaded to our servers in Guest mode.
  • Risk: clearing site data or switching devices will lose your chats and key.
Reminder: API keys in the browser can be read by any script on the page. Use trusted devices.

Google Sign-In

Sign in to sync across devices. Your API key is encrypted client-side before it’s saved to Firebase. Chat content is saved in the cloud as plain text for multi-device access and recovery.

  • Keys: encrypted in the browser (AES-GCM) and stored in Firebase per account.
  • Chats: stored per user; deleting a chat removes it from the cloud.
  • Transit: all network traffic uses HTTPS/TLS.
Tip: prefer Guest mode for sensitive conversations you don’t want in the cloud.

How to Use

  1. Sign in with Google or continue as Guest.
  2. Enter your API key in Settings/Sidebar (we never provide keys).
  3. Select a model (OpenAI / DeepSeek / Gemini).
  4. Attach files (optional). Text/code is added to your prompt; images on vision models.
  5. Send your message. Use Stop to cancel generation.
  6. Use the chat ⋯ menu to rename, export, import into chat, or delete.

Privacy Details

Data We Store

  • Guest: API key + chats only on your device. We do not see or collect them.
  • Google: API key encrypted client-side in Firebase; chats stored per user (plain text).

Retention & Deletion

  • Guest: clearing browser/site data deletes everything permanently.
  • Google: delete chats from the app to remove them from the cloud; logout clears local caches.

Third-Party Providers

  • You connect directly to OpenAI/DeepSeek/Gemini using your key; their terms and pricing apply.
  • We don’t resell keys. Keep your key private.

Limits, Costs & Safety

  • Token limits: very long chats are auto-trimmed from the oldest turns.
  • Rates & fees: usage is billed to your own provider account.
  • Images: only models with vision can analyze images; others will ignore image content.
